File > New > Java Project
JRE - Use an execution environment JRE => JavaSE-1.6
In new project folder, create 'lib' subfolder, create 'models' subfolder
Into libs: goes the extracted files downloaded from here: http://opennlp.apache.org/cgi-bin/download.cgi
- jwnl-1.3.3.jar
- opennlp-maxent-3.0.3.jar
- opennlp-tools-1.5.3.jar
- opennlp-uima-1.5.3.jar

Into models goes the models (.bin) files downloaded from here: http://opennlp.sourceforge.net/models-1.5/

Create new class in project
- add method stub: public static void main(String[] args)

Check: http://www.programcreek.com/2012/05/opennlp-tutorial/
